# Unicode as Cognitive Input — Proof v1

## إثبات اليونيكود كمعطى عقلي أولي

---

## الدعوى المركزية

> **أن البرنامج يستلم المعطى من اليونيكود كمعطًى عقلي أولي، لا لأنه معنى مكتمل،
> بل لأنه مادة قابلة للتعيين والتمييز والتصور؛ ثم يعيد بناء هذا المعطى اليونيكودي
> نفسه داخل سلسلة العقل، طبقةً بعد طبقة، وفق أوليات العقل والحد الأدنى المكتمل
> لكل طبقة، ومن غير قفز، حتى يبلغ رتبة المعطى العقلي المحرر القابل للدلالة ثم للحكم.**

---

## 1. تعريف المعطى العقلي الأولي (المادة 1)

المعطى العقلي الأولي هو:

> كل مادة تدخل في مجال العقل من حيث التلقي والتمثيل الأولي، من غير أن تكون
> بذلك قد صارت معنىً مكتملًا أو موضوعًا محررًا أو حكمًا منجزًا.

## 2. تعريف Unicode في هذا المقام (المادة 2)

> تمثيل رقمي أولي لوحدات نصية قابلة للالتقاط والتعيين والتصنيف داخل النظام.

## 3. الحكم (المادة 3)

لا يعامل Unicode في هذا المشروع بوصفه معنىً مكتملًا ولا لفظًا تامًا ولا مفردًا
معتمدًا ولا حكمًا، بل بوصفه:

> مادة ترميزية أولية قابلة لأن تدخل في سلسلة العقل بوصفها معطًى قابلًا
> للتعيين والتمييز والتصور.

---

## 4. السلسلة العامة لإعادة العقلنة (المادة 9)

```text
U₀  Unicode Raw Input     → المدخل اليونيكودي الخام
U₁  Atomized Input         → المُعطى المُعيَّن ذريًا
U₂  Differentiated Units   → الذرّات المتميزة المتشاكلة
U₃  Normalized Units       → المُعطى المُطبَّع
U₄  Designated Units       → المُعطى الحاضر المفروق المُعيَّن
U₅  Initial Conception     → التصور الأولي
U₆  Disciplined Conception → التصور المنضبط
U₇  Semantic Subject       → الموضوع الدلالي المحرر
U₈  Judgement-Ready Input  → المُعطى الصالح للحكم
```

---

## 5. الحد الأدنى المكتمل لكل طبقة

### U₀ → U₁: الذرية (المادة 12–14)
1. التقاط الوحدة
2. تعيينها كذرة
3. تصنيفها الأولي (`SignalType`)
4. إسناد وظيفة أولية
5. الحكم بصلاحيتها أو تعليقها أو ردها

### U₁ → U₂: التمييز والتشاكل (المادة 15–17)
1. تمييز الذرة عما يجاورها
2. تعيين نوعها
3. تعيين تشاكلها الأولي
4. بيان قابليتها للانتظام مع غيرها

### U₂ → U₃: التجميع والتطبيع (المادة 18–20)
1. تجميع الذرات في عناقيد معتبرة
2. حفظ الأصل الخام
3. إنشاء صورة معيارية للعمل
4. منع ضياع المعلومة المعتبرة
5. تسجيل سياسة التطبيع

### U₃ → U₄: الحضور والفرق والتعيين (المادة 21–25)
1. الحضور — حاضر للعقل
2. الفرق — متميز عما ليس هو
3. التعيين الأولي — جهة أولية
4. القابلية لأن يُبنى عليه تصور

### U₄ → U₅: التصور الأولي (المادة 26, 28)
1. وحدة أولية
2. حد أولي
3. جهة أولية
4. إمكان الاستحضار

### U₅ → U₆: التصور المنضبط (المادة 27, 29)
1. ضبط الحد
2. ضبط الوحدة
3. إزالة الاشتباه المفسد
4. صلاحية الانتقال إلى الترميز التصوري

### U₆ → U₇: الترميز والدلالة (المادة 31–33)
- الترميز التصوري بعد التصور
- الدلالة المحررة

### U₇ → U₈: تحرير الموضوع والحكم
- تحرير الموضوع
- تعيين جهة الحكم
- الصلاحية للحكم

---

## 6. قانون منع القفز (المادة 34–37)

### القفزات الممنوعة (المادة 35):
- Unicode → معنى مكتمل
- Unicode → مفرد معتمد
- Unicode → مفهوم
- Unicode → حكم
- تطبيع → حكم
- تصور أولي → حكم
- ترميز تصوري → حكم بلا تحرير موضوع

### القاعدة (المادة 37):
> لا يجوز الانتقال من Unicode إلى الحكم إلا عبر الطبقات التي تعيد عقلنته
> طبقةً بعد طبقة حتى يبلغ رتبة الموضوع المحرر القابل للدلالة ثم للحكم.

## 7. الصياغة الرياضية (المادة 41–46)

### التمثيل العام (المادة 41):

$$U_0 \to U_1 \to U_2 \to U_3 \to U_4 \to U_5 \to U_6 \to U_7 \to U_8$$

### شرط الانتماء (المادة 42):

$$C_i(x) \in \{0,1\},\quad x \in U_i \iff C_i(x)=1$$

### شرط الحد الأدنى المكتمل (المادة 43):

$$M_i(x) \in \mathbb{R},\quad M_i(x) \ge \theta_i$$

### دالة العبور (المادة 44):

$$T_i: U_i \to U_{i+1} \cup \{\text{Reject}, \text{Suspend}, \text{Complete}\}$$

### شرط العبور العام (المادة 45):

$$C_i(x)=1 \land M_i(x)\ge\theta_i \land \neg F_i(x)$$

## 8. البرهان العام (المادة 47–52)

### المقدمات:
1. **المادة 47**: Unicode يدخل النظام أولًا بوصفه مادة ترميزية قابلة للالتقاط
2. **المادة 48**: المادة القابلة للالتقاط ليست معنىً مكتملًا لكنها قابلة للتعيين والتمييز
3. **المادة 49**: فيجوز معاملة Unicode بوصفه معطًى عقليًا أوليًا
4. **المادة 50**: المعطى الأولي لا يصير صالحًا للدلالة إلا بعد إعادة بنائه عبر طبقات
5. **المادة 51**: لكل طبقة حد أدنى مكتمل → يمتنع القفز

### النتيجة (المادة 52):
> يثبت أن البرنامج يستلم المعطى من Unicode كمعطًى عقلي أولي، ثم يعيد بناءه
> داخل سلسلة العقل وفق أوليات العقل والحد الأدنى المكتمل لكل طبقة، ومن غير قفز،
> حتى يبلغ رتبة المعطى العقلي المحرر القابل للدلالة ثم للحكم.
